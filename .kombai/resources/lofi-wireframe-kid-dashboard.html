Design Notes

  • No AI references — removed HeyGen avatar, TutorPreviewCard, AvatarErrorBoundary
  • Landing page aesthetic — cream off-white (#f4efe5), thin 1px borders, tracked uppercase labels, 300 font-weight headings, pink accent (#c8416a)
  • Editorial typography — italic accent in headings, section numbers in pink, uppercase metadata labels
  • 3-column action grid (was 2-column) — better use of horizontal space on desktop
  • Consistent card style — all cards use same border-radius (4px), same border, same background
  • Bottom nav updated — backdrop-blur matching the landing page header style, renamed "Ask Tutor" to "Study"
  • All new CSS classes added to vertex.css with vtx-kid-* prefix for clean separation
